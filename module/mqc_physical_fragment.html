Physical molecular fragment representation and geometry handling

Provides data structures and utilities for managing molecular fragments with atomic coordinates, electronic properties, and geometric operations. Single molecular fragment type Complete system geometry type System geometry initialization Extract fragment from system Build fragment from explicit atom indices (for intersections) Validate fragment has no overlapping atoms
